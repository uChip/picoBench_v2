## picoBench° desktop application
An application for displaying probe values and controlling the picoBench° hardware.

### Features

General
	USB interface selectable COM port

Power Supply
	Control enable / disable
	PS output voltage display
	PS current display
	Log PS voltage and current data to a file

Voltage Probe
	3.5 digit voltage display
	Select resolution of 1 mV or 100 uV
	Sample instantanious or average voltage
	Log voltage to a file
	Chart voltage over time
	
Current Probe
	3.5 digit amps display
	Select resolution of 1 mA or 10 uA
	Sample instantanious or average amps
	Log amps to a file
	Chart amps over time
	
### File Formats

This is a MS Visual Studio project written in Visual Basic.
